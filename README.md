# B-6

B-6 is a simple music game running in static page with randomly generated art work as background.

Using color-wander as wheel for background art.

## Usage

Open the demo [here](https://bee-6.com).

First click any where on the acreen to start the game. Be careful when you start game there will be music playing. Don't play it in class :)

After game started, there will be black circle in the middle of screen. Your job is to tap screen or press space when the inner solid circle reach the outter arc. The closer solid circle is to the arc, the higher score you will get!

Enjoy your game!

## Technology
We used Vanilla JS and HTML 5 to build this game.
